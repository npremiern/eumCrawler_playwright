🧪 테스트 방법

GUI에서 테스트

  1. crawler_gui.py 실행
  2. Excel 파일 선택
  3. 축적 선택: 드롭다운에서 1/3000 또는 1/6000 선택
  4. PDF 다운로드: 체크박스 활성화 ✓
  5. 크롤링 시작
  6. pdfs/ 폴더에서 결과 확인

CLI에서 테스트

# 1/3000 축적으로 PDF 저장 python src/crawler.py -f example_data.xlsx --scale 3000 # 1/6000 축적으로 PDF 저장 python src/crawler.py -f example_data.xlsx --scale 6000
🔧 최근 수정 사항 (2026-02-12)

1. URL 수정 FIXED

  • 이전: {BASE_URL}/luLandPop.jsp?pnu={pnu}&scale={scale}
  • 수정: https://www.eum.go.kr/web/ar/lu/luLandPop.jsp?pnu={pnu}&sMode=search&default_scale=3000&scale={scale}
  • 이유: 이미지 다운로드와 동일한 URL 형식 사용

2. 파일명에 축적 추가 NEW

  • 이전: {ID}_{주소}.pdf
  • 수정: {ID}_{주소}_{축적}.pdf
  • 이유: 어떤 축적으로 저장되었는지 파일명으로 구분 가능

✅ 결론: 축적을 변경하면 자동으로 팝업창을 사용하여 변경된 축적이 PDF에 정확히 반영됩니다!
