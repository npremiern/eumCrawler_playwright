==============================================
부동산 정보 크롤링 프로그램 v1.0
Real Estate Information Crawler
==============================================

[프로그램 소개]
Excel 파일에서 주소를 읽어 국토교통부 부동산공시가격 알리미
사이트(eum.go.kr)에서 부동산 정보를 자동으로 크롤링하여
Excel 파일에 기록하는 프로그램입니다.

[사용 방법]
1. Excel 파일의 B열에 검색할 주소를 입력하세요
   (예: 서울특별시 강남구 테헤란로 152)

2. 명령 프롬프트(cmd)를 열고 다음 명령어를 실행하세요:
   crawler.exe --file "파일경로\data.xlsx"

   예시:
   crawler.exe --file "C:\Users\Username\Desktop\data.xlsx"

3. 프로그램이 자동으로 각 주소를 검색하고 정보를 저장합니다.

[Excel 파일 구조]
- B열: 검색할 주소 (입력)
- C열: 주소 (출력)
- D열: 분류 (출력)
- E열: 면적 (출력)
- F열: 지가 (출력)
- G열: 표시1 (출력)
- H열: 표시2 (출력)
- I열: 표시3 (출력)
- J열: 이미지 (출력)

[추가 옵션]
--start-row 3        # 3행부터 시작
--no-headless        # 브라우저 창 표시
--verbose            # 상세 정보 출력
--wait 5             # 대기 시간 5초로 설정

예시:
crawler.exe --file data.xlsx --start-row 3 --verbose

[필수 요구사항]
- Windows 10/11 (64-bit)
- 인터넷 연결
- 최소 4GB RAM
- 최소 500MB 디스크 여유 공간

[첫 실행 시]
✓ 자동으로 Chromium 브라우저를 다운로드합니다
  - 크기: 약 200MB
  - 소요 시간: 3-5분
  - 인터넷 연결 필수
  - 설치는 최초 1회만 진행됩니다

⚠ 만약 자동 설치가 실패하면:
  1. 명령 프롬프트를 관리자 권한으로 실행
  2. 다음 명령어 입력:
     playwright install chromium

자세한 설치 가이드는 INSTALL_GUIDE.txt를 참고하세요.

[문제 해결]
Q: "파일을 찾을 수 없습니다" 오류가 발생해요
A: Excel 파일 경로가 정확한지 확인하세요.
   경로에 공백이 있으면 따옴표로 감싸세요.

Q: 백신 프로그램이 차단해요
A: 백신 프로그램의 예외 항목에 crawler.exe를 추가하세요.
   이 프로그램은 안전합니다.

Q: "DLL 로드 실패" 오류가 발생해요
A: Visual C++ 재배포 패키지를 설치하세요.
   https://aka.ms/vs/17/release/vc_redist.x64.exe

Q: 프로그램이 느려요
A: --wait 옵션으로 대기 시간을 늘려보세요.
   예: crawler.exe --file data.xlsx --wait 5

Q: 특정 주소만 검색 결과가 없어요
A: 해당 주소가 부동산공시가격 알리미에 등록되지
   않았을 수 있습니다. 주소를 정확히 입력했는지
   확인하세요.

[주의사항]
- 프로그램 실행 중 Excel 파일을 열지 마세요
- 안정적인 인터넷 연결이 필요합니다
- 대량의 주소 처리 시 시간이 걸릴 수 있습니다
  (100개 주소 기준 약 10-15분)
- 웹사이트 이용 약관을 준수해주세요
